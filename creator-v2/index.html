Text Codes:
Code
Result

Inserts the name of the card (or use a tilde: ~)

Inserts an em-dash
{i}
Italicizes text

Removes italicization
{bold}
Bolds text
{/bold}
Removes bold
{lns}
Moves to the next line without an extra space
{divider}
Moves to the next line and draws the flavor text bar

Moves to the next line, draws the flavor text bar, and italicizes

Italicizes and adds line breaks

Changes the font size to #pt (absolute)

How to find set codes
Set codes are the two-three character combinations representing sets. For sets released after 2015, the three-character code is in the lower left corner.

Set symbols use Scryfall codes. Watermarks use Keyrune codes.

How are my cards saved?
Cards are saved in your browser's localStorage (usually up to 5MB). Uploading images via URL instead of file upload saves significant space.
